# Fire-Extinguisher-Planner
Fire Extinguisher Planner based on NFPA 10


**Fire Extinguisher Planner** is a professional desktop application designed to assist safety engineers, architects, and HSE professionals in the optimal placement and calculation of portable fire extinguishers according to **NFPA 10 (2022 Edition)** standards.

This tool simplifies the complex process of calculating travel distances, hazard levels, and area coverage using a CAD-like drawing interface and automated geometric analysis.

## โจ Key Features

  * **CAD Drawing Interface:** Draw floor plans directly within the software or define boundaries.
  * **NFPA 10 Compliance:** Automatically calculates requirements based on Hazard Class (Light, Ordinary, Extra) and Hazard Type (Class A, B, C, D, K).
  * **Advanced Coverage Analysis:**
      * Calculates **Travel Distance** limits.
      * Includes a **Safety Factor** input to account for obstacles (walls, furniture) impacting the actual walking path vs. geometric radius.
  * **Smart Placement Algorithms:** Generates multiple placement scenarios (Grid, Offset, Hexagonal) to minimize the number of units while ensuring full coverage.
  * **Print Report:** Exports detailed **PDF Reports** including the coverage map, calculation breakdown, and compliance status.
  * **Dual Unit Support:** Full support for both **Imperial (ft/sq ft)** and **Metric (m/sq m)** systems.
